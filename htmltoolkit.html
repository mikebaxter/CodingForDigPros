CFDP's HTML Toolkit

Semantic elements

  • <h1> - main header on page
  • <h2> ‐ <h6> - progressive sub-headers
  • <p> - paragraph
  • <strong> - makes text stronger (like bold)
  • <em> - gives text emphasis (like italics)

Structural elements

  • <span> - inline element
  • <div> - block element

Other elements

  • <hr> - a horizontal line (rule)
  • <img> - an image; requires a source attribute (src="image url")
  • <a> - a link; requires a link target (href="target url")
  • Note the safest way to add an image or target URL is to navigate, in your browser, to the web page or image you want to link to and then copy and paste its URL.
