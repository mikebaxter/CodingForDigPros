Client-Server Architecture

How the web works

The web works by means of request:response communication between client and server devices.

When a link is clicked (or a URL typed in a browser address bar) the client sends a request for the relevant web page and the server responds either by sending back the requested resource or with an error message indicating why the resource was not available (e.g. 404 error page)
